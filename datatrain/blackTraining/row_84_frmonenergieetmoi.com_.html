Important Notice: Android App Compatibility

The new Beta version of the Android app is not compatible with the old (stable) version.
Likewise, the old version is not compatible with the Beta version.

  • If you want to use the Beta version, uninstall the old version and download the Beta Android app from:
    https://rbsoft.org/downloads/sms-gateway/sms-gateway-beta.apk
  • If you prefer the Stable version, uninstall the Beta version and download the Stable Android app from:
    https://rbsoft.org/downloads/sms-gateway/sms-gateway.apk
To avoid issues, please ensure you uninstall the version you are currently using before installing the other.

Customize Android Application

You can create a customized version of the Android app using our build server.

  1. Create an account by using your purchase code here:
    https://app-builder.rbsoft.org/register
    (You can skip this step if you already have an account.)
  2. After registering, login to the account you just created and follow the step-by-step video tutorial:
    https://youtu.be/NyC0Ipji5_s
  • For the Stable version, choose SMS Gateway Legacy as the Project on the build server.
  • For the Beta version, choose SMS Gateway.
Custom builds for the Beta version are not yet available, but support is coming soon.

Thank you for your understanding.
